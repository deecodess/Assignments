
## Numerical Analysis
This repository contains MATLAB code for the following topics:

 - Root Finding Methods:
		 1. Bisection
		 2. Secant
		 3. Newton Raphson
		 4. Fixed Point Iteration
	
 - System of Equations
		 1. Gauss Seidal
		 2. Gauss Elimination
		 3. LU Decomposition
 - Interpolation
		 1. Lagrange Interpolation
		 2. Newton Divided
 - Numerical Integration
		 1. Trapezoidal
		 2. Simpsons one-third
 - Numerical Differentiation
		 1. Euler
		 2. Runge Kutta 4th order
 - Power Method
